# Getting Started

To get started with using Sessions Bot for the first time please instal the bot to any Discord server you <u>manage</u> or <u>own</u>.

### Who is it available for?

> The bot is available to any valid Discord user at absolutely no cost! However, premium features are available through purchase.

## First Steps:

1. [Invite the Bot](https://invite.sessionsbot.fyi) to a Discord server you _manage or own_.
2. Grant Sessions Bot with all of it's requested permissions upon invite.

    > Related: [`Bot Permissions`](/bot-permissions)

3. Visit your [Bot Dashboard](https://sessionsbot.fyi/dashboard) to start using Sessions Bot and it's features! ðŸŽ‰
